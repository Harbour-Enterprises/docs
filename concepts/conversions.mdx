## How Document Conversion Works

SuperDoc uses Gotenberg, a powerful document conversion engine, to transform DOCX files into PDFs while preserving formatting, styles, and layout.

## Conversion Process

<Steps>
  <Step title="Upload">Your DOCX file is uploaded to our secure servers</Step>
  <Step title="Validation">
    We validate the file format and check for compatibility
  </Step>
  <Step title="Conversion">
    Gotenberg processes the document using LibreOffice
  </Step>
  <Step title="Optimization">The PDF is optimized for size and quality</Step>
  <Step title="Delivery">The converted PDF is returned in the response</Step>
</Steps>

## Best Practices

### Optimize Your Documents

<Tip>
  For best results: - Use standard fonts (Arial, Times New Roman, Calibri) -
  Embed custom fonts in your DOCX - Keep file size under 10MB for faster
  processing - Use high-resolution images (300 DPI)
</Tip>

### Handling Large Files

For files larger than 25MB:

1. **Compress images** before conversion
2. **Split documents** into smaller parts
3. **Contact support** for enterprise options

## Troubleshooting

<Warning>Common issues and solutions:</Warning>

### Fonts Not Displaying Correctly

**Problem**: Custom fonts appear as default fonts in PDF

**Solution**:

- Embed fonts in your DOCX file
- Use standard system fonts
- Convert fonts to curves/outlines

### Images Missing or Distorted

**Problem**: Images don't appear or look wrong

**Solution**:

- Use standard image formats (JPEG, PNG)
- Ensure images are embedded, not linked
- Check image resolution (300 DPI recommended)

### Layout Issues

**Problem**: Page breaks or margins are different

**Solution**:

- Use explicit page breaks
- Set fixed margins in DOCX
- Avoid complex nested tables
