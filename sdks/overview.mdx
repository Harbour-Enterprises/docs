## Common Features

All SuperDoc SDKs include:

### Core Functionality

- **Document conversion** - DOCX to PDF conversion
- **File handling** - Support for local files, streams, and URLs
- **Error handling** - Comprehensive error types and messages
- **Authentication** - Automatic API key management

### Advanced Features

- **Retry logic** - Automatic retries with exponential backoff
- **Rate limiting** - Built-in rate limit awareness
- **Progress tracking** - Upload and conversion progress callbacks
- **Webhooks** - Signature verification for webhook endpoints

### Configuration Options

- **API endpoints** - Custom base URLs for enterprise customers
- **Timeouts** - Configurable request and processing timeouts
- **Proxy support** - HTTP/HTTPS proxy configuration
- **Logging** - Configurable logging levels and outputs

## Quick Start Examples

<CodeGroup>
```javascript JavaScript/Node.js
import { SuperDoc } from '@superdoc/api';

const client = new SuperDoc('sd_live_pk_your_api_key');

const pdfBuffer = await client.convert({
file: './document.docx',
format: 'pdf'
});

````

```python Python
from superdoc import SuperDoc

client = SuperDoc('sd_live_pk_your_api_key')

pdf_content = await client.convert(
    file='./document.docx',
    format='pdf'
)
````

```php PHP (Coming Soon)
use SuperDoc\Client;

$client = new Client('sd_live_pk_your_api_key');

$pdf = $client->convert([
    'file' => './document.docx',
    'format' => 'pdf'
]);
```

```go Go (Coming Soon)
package main

import "github.com/superdoc/go-sdk"

client := superdoc.New("sd_live_pk_your_api_key")

pdf, err := client.Convert(superdoc.ConvertOptions{
    File:   "./document.docx",
    Format: "pdf",
})
```

</CodeGroup>
